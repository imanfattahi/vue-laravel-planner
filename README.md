## Installation

 - Clone project
 - Go to project folder
 - Run "npm install"
 - Run "php artisan serve"
 - Run "npm run watch" or "npm run dev" 

## Test

 - Run "vendor/phpunit/phpunit/phpunit" Or install phpunit and run "phpunit"
